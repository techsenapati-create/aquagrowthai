# AquaGrowthAI

ICAR-CIFA Research Prototype

Machine Learning Based Growth Stage Classification and Future Growth Forecasting for Indian Major Carps.

## Dataset

- 639 observations
- 9 months monitoring (Oct to June)
- Male and Female fish

## Models

### Classification

Random Forest Classifier

Accuracy: 49.22%

### Forecasting

Random Forest Regressors

| Parameter | R² |
|------------|------------|
| Length | 0.9549 |
| Weight | 0.9721 |
| Depth | 0.8928 |
| DPL | 0.9275 |
| Perimeter | 0.9383 |
