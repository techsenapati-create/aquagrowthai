# AquaGrowthAI

ICAR-CIFA Research Prototype

Machine Learning Based Growth Stage Classification and Future Growth Forecasting for Indian Major Carps.

## Dataset

- 492 observations
- 8 months monitoring
- Male and Female fish

## Models

### Classification

Random Forest Classifier

Accuracy: 66.67%

### Forecasting

Random Forest Regressors

| Parameter | R² |
|------------|------------|
| Length | 0.9582 |
| Weight | 0.9704 |
| Depth | 0.9268 |
| DLPL | 0.9181 |
| Perimeter | 0.9571 |
